
# ORTO CORE

## Overview
This repository contains multiple microservices along with the necessary CI/CD pipelines, infrastructure as code, deployment scripts, and monitoring configurations.

## Prerequisites
- Docker
- Kubernetes
- Terraform
- AWS CLI
